# ToDo List


### By:

---

**_Description_**

- A To Do list that allows users to add, update, and delete

---

**_Getting Started_**
- Git Clone branch
- cd into todolist
- run npm run server
- then open a new terminal
- run npm run client
- View page in http://localhost:3000

---
